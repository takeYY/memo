# memo

メモを記載する

https://takeyy.github.io/memo/#/

## 使い方

1. docsify-cli のインストール

   !> docsify コマンドが使えない人のみ

   ```bash
   npm i docsify-cli -g
   ```

2. docsify を初期化
   ```bash
   docsify init .
   # . already exists.
   # ✔ Are you sure you want to rewrite it? (y/N)
   y
   ```
3. 動作確認
   ```bash
   docsify serve .
   ```
4. Let's Enjoy!
